# 掌上华医 AutoX.js 全自动刷课 + 智能考试脚本
## 脚本介绍
这是一款基于 **AutoX.js** 开发的**掌上华医全自动学习+自动考试脚本**，无需手动操作，自动完成视频播放、课堂思考答题、课程遍历学习、智能答题、自动建题库、自动重考，真正解放双手，高效完成学习与考试任务。

---
## 核心功能
### 📺 全自动刷课
1. **全自动播放视频**
   自动检测播放进度、唤醒进度条、等待视频播放完成，实时显示播放百分比。
2. **智能处理课堂思考**
   自动识别课堂问答，交替选项答题，自动完成直至正确。
3. **批量遍历课程**
   自动识别「未学习/播放至」课程，逐个打开学习，无需手动干预。
4. **自动导航**
   启动APP → 我的 → 我的收藏，一键开始全自动学习。
5. **静音运行**
   自动静音，学习完成恢复原音量，不打扰使用。

### 📝 全自动智能考试（独家核心）
1. **自动识别待考试课程**
   自动遍历收藏中所有「待考试」课程，自动进入考试。
2. **智能题库系统**
   自动记录题目与正确答案，本地保存题库，下次直接选正确答案。
3. **自动答题逻辑**
   - 题库有答案 → 自动选择正确答案
   - 题库无答案 → 按轮次选择 A→B→C→D→E
4. **自动判题 + 自动建库**
   考试未通过 → 自动识别绿色对勾/红色叉号 → 提取正确答案 → 写入本地题库。
5. **自动重考机制**
   考试未通过自动重新考试，最多自动重考 **5 次**，直至通过。
6. **自动交卷 + 自动继续**
   答题完毕自动交卷，通过后自动继续下一门课程考试。

## 使用环境
- 设备：安卓手机（无需Root）
- 运行工具：**AutoX.js**（推荐最新版）
- 适用APP：掌上华医 V3.135.10
- 脚本语言：JavaScript
- 支持题型：单选题

---
## 使用步骤
### 1. 准备工作
1. 安装 **AutoX.js** 并开启**无障碍服务**
2. 安装「掌上华医」并登录账号
3. 将需要学习/考试的课程**加入我的收藏**
4. 首次运行授权：**无障碍权限、截图权限**

### 2. 运行脚本
1. 打开 AutoX.js
2. 新建脚本 → 粘贴代码
3. 点击运行 → 授权权限
4. 脚本全自动启动华医、刷课、考试

### 3. 运行效果
- 控制台实时打印：课程进度、题目内容、答题结果、考试次数
- 自动答题、自动判题、自动记录答案
- 考试通过自动继续下一门
- 全程无需看管

---
## 脚本说明
### 刷课逻辑
- 超时保护：单视频最大等待 4000 秒
- 进度判断：根据播放进度判断是否完成
- 自动返回：完成后自动返回课程列表继续下一个

### 考试逻辑
- 题目清洗：自动去除「单选1、、2、、3.」等前缀，统一题目格式
- 本地题库：`考试题库.json` 保存在**脚本同一目录**，可查看、备份
- 重考规则：A→B→C→D→E，最多 5 轮
- 判题方式：截图识别绿色对勾=正确，红色叉号=错误

---
## 注意事项
1. 首次运行必须开启：**无障碍权限 + 截图权限**
2. 课程必须放在「我的收藏」中
3. 运行时尽量不要手动操作手机，避免界面错乱
4. 华医APP更新可能导致控件ID失效，需同步更新ID
5. 脚本仅供学习交流，请勿用于商业用途
6. 建议在固定页面运行脚本，不要在后台杀死APP
7. 打开视频检测到当前为移动网络，会自动点击继续，请在WiFi环境下运行，避免浪费流量。

## 目前BUG
1. 打开掌上华医偶尔会出现开屏广告/更新弹窗，遇到需手动关闭
2. 部分特殊界面题目格式可能需要微调清洗规则
3. 不支持复杂题型（如不定项、拖拽题）
4. 不要收藏含互动病例的视频，或者做完后再收藏（因为并没有针对此情况写代码,懒得改...）
