# AutoX.js-Ai-huayi-
# 掌上华医 AutoX.js 自动刷课脚本
## 脚本介绍
这是一款基于 **AutoX.js** 开发的**掌上华医全自动学习脚本**，无需手动操作，自动完成视频播放、课堂思考答题、课程遍历学习，解放双手，高效完成学习任务。

---
## 核心功能
1. **全自动播放视频**
   自动检测播放进度、弹出进度条、等待视频播放完成，支持百分比进度实时显示。
2. **智能处理课堂思考**
   自动识别课堂问答，优先选择第一个选项提交，错误自动切换第二个选项，自动完成答题。
3. **批量遍历课程**
   自动识别「未学习/播放至」状态的课程，逐个打开学习，无需手动点击。
4. **自动导航**
   启动APP → 自动进入「我的」→「我的收藏」，一键开始学习。
5. **安全防崩溃**
   异常捕获、超时保护、控件失效自动刷新，脚本稳定运行不闪退。
6. **静音学习**
   自动开启静音，学习完成后恢复原音量，不打扰使用。

---

## 使用环境
- 设备：安卓手机（无需Root）
- 运行工具：**AutoX.js**（推荐最新版）
- 适用APP：掌上华医
- 脚本语言：JavaScript

---

## 使用步骤
### 1. 准备工作
1. 手机安装 **AutoX.js**
2. 开启 AutoX.js 的**无障碍服务**
3. 将掌上华医想要学习的课程添加到**我的收藏**（脚本从收藏页开始学习）

### 2. 运行脚本
1. 打开 AutoX.js
2. 新建脚本，粘贴本代码
3. 点击运行，授权无障碍权限
4. 脚本自动启动掌上华医，开始全自动学习

### 3. 运行效果
- 控制台实时打印：等待时间、播放进度、课程完成状态
- 自动处理弹窗、答题、返回、切换课程
- 全部完成后自动停止、恢复音量、关闭控制台

---

## 脚本说明
1. **进度计算**
   实时计算播放百分比，格式：`当前进度：68.5% (02:30/05:00)`
2. **超时保护**
   单视频最大等待 4000 秒，超时自动退出
3. **课堂思考逻辑**
   先选第1个选项 → 提交 → 仍存在题目则选第2个 → 再次提交
4. **遍历规则**
   自动识别所有「未学习/播放至」课程，逐个完成
5. **安全退出**
   学习完成自动恢复音量、关闭线程、停止所有脚本

---

## 注意事项
1. 首次运行必须开启 **AutoX.js 无障碍权限**
2. 请确保掌上华医已登录，且课程在「我的收藏」中
3. 运行时请勿手动操作手机，避免干扰脚本
4. APP更新可能导致控件ID失效，如无法运行需更新对应ID
5. 本脚本仅供学习交流，请勿用于商业用途

## 目前BUG
1. 如果看的是已经完成的课程，如果视频时间长度过短会不停重复观看此次视频，直到4000 秒后，才会超时自动退出。原因是需要点击视频唤醒进度条获取时间，如果视频已经播放完状态，会变成重新开始播放视频。
2. 打开掌上华医有时会有弹窗，但暂时还没遇到，所以需要手动跳过。
3. 如果课堂问答选项开始是选中状态，则无法完成答题。
